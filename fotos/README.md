# Diretório de Evidências Fotográficas

Cole neste diretório as fotos tiradas durante as oficinas práticas realizadas na ONG (mínimo de 2 a 3 fotos para fins de comprovação da atividade de extensão da Estácio).

### Recomendações:
- Formatos sugeridos: `.jpg`, `.jpeg` ou `.png`
- Nomes sugeridos para os arquivos:
  - `foto_oficina_1.jpg` (Momento da explicação sobre letramento digital)
  - `foto_oficina_2.jpg` (Participantes utilizando o gerador de currículos no computador)
  - `foto_oficina_3.jpg` (Roda de conversa final e entrega de material)

*Observação: Lembre-se de compactar este diretório como `fotos_oficina_inclusao.zip` caso o portal acadêmico da Estácio exija envio de arquivo único compactado, conforme citado no relatório.*
